# AuthApp 🔐

App con autenticación completa construida con React Native y Expo.

![AuthApp Register](https://raw.githubusercontent.com/fer1Devapp/AuthApp/main/Register.png)

## Funcionalidades
- 🔑 Login con validación
- 📝 Registro de usuarios
- 💾 Sesión persistente con AsyncStorage
- 🔒 Rutas protegidas
- 🚪 Cerrar sesión

## Tecnologías
- React Native
- Expo
- TypeScript
- React Navigation
- Context API
- AsyncStorage

## Cómo correrlo
```bash
git clone https://github.com/TuUsuario/AuthApp.git
cd AuthApp
npm install
npx expo start
```
